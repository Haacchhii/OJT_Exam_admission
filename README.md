# Golden Key Integrated School — Online Exam & Admission System

A full-stack web application for managing entrance exams and student admissions.

## Project Structure

```
├── frontend/    # React + Vite + Tailwind CSS v4
│   ├── src/
│   ├── package.json
│   └── ...
├── backend/     # Server API (coming soon)
└── README.md
```

## Frontend

```bash
cd frontend
npm install
npm run dev      # http://localhost:5173
```

Set `VITE_API_URL` in `frontend/.env` to connect to the backend.  
Leave it empty to use the built-in localStorage demo mode.

See [frontend/API_CONTRACT.md](frontend/API_CONTRACT.md) for the full API specification.
